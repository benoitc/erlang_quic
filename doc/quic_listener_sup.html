    quic_listener_sup (quic v0.7.0)

    View Source

    Supervisor for a pool of QUIC listeners using SO_REUSEPORT.

    This module provides horizontal scaling for QUIC servers by running multiple listener processes that share the same port via reuseport. The kernel distributes incoming packets across the listeners.

    Usage

    Single listener (default):

       quic_listener:start_link(Port, Opts)

    Pooled listeners for scalability:

       quic_listener_sup:start_link(Port, Opts#{pool_size => 4})

    See also: quic_listener.
